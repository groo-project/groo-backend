## 서비스 설명

🌱 우리는 하루에도 여러 감정을 겪지만, 그 감정들이 지나간 자리에는 종종 아무것도 남지 않습니다.<br>
🌱 Groo는 바로 이 순간들을 놓치지 않고, 감정을 기록하고 표현할 수 있는 특별한 공간을 만들고자 시작되었습니다.

단순한 텍스트 일기장을 넘어, Groo는 감정을 시각적으로 표현합니다.<br>
사용자가 일기를 작성하면, AI가 감정을 분석하고 그 결과에 따라 정원을 꾸밀 수 있는 자연 요소(꽃, 나무, 날씨 등)를 제공합니다.<br>
이렇게 표현된 감정은 ‘하루치 마음’을 담은 작은 정원이 되고, 시간이 지날수록 하나의 개인적인 감정의 숲으로 확장되어 갑니다.
